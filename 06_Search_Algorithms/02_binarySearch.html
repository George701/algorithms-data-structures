  • Binary search only workds on sorted arrays!
  • Divide and Conquer!

Binary Search Pseudocode

  • Write function that accepts a sorted array and a value
  • Create a left pointer at the start of the array, and a right pointer at the end of the array
  • While the left ponter comes before the right pointer:
    • Create a ponter in the middle
    • If you find the value you want, return the index
    • If the value is too small, move the left pointer up
    • If the value is too large, move the left pointer down
  • If you never find the value, return -1
