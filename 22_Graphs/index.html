Graphs

Graph is a collection of nodes and connections.

A graph data structure consists of a finite (and possibly mutable) set of vertices or nodes or points, together with a set of unordered pairs of these vertices for an undirected graph or a set of ordered pairs for a directed graph.

Adding a vertex

  • Write a method called addVertex, which accepts the name vertex
  • It should add a key to the adjacency list with the name of the vertex and set its value to be an empty array

Add an edge

  • Function must accept two vertices
  • The function should find in the adjacency list the key of first vertex and push second vertex to the array
  • The function should find in the adjacency list the key of second vertex and push first vertex to the array
